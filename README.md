# Student-Marks-Prediction-using-ML-Linear-Regression-Model-Python-and-Flask
A Flask-based machine learning web application that predicts student marks based on study hours using Simple Linear Regression, featuring a modern and responsive user interface.

📚 Student Marks Prediction

A simple Machine Learning web application that predicts student marks based on the number of hours studied using Simple Linear Regression.

🚀 Features

* Predict marks from study hours
* Flask-powered web application
* Modern responsive UI with glassmorphism design
* Beginner-friendly ML project

🛠 Tech Stack

* Python
* Flask
* Scikit-learn
* HTML5
* CSS3

📂 Project Structure

├── main.py
├── index.html
├── style.css
├── Simple Linear Regression.ipynb
└── README.md


Open:
http://127.0.0.1:5000/

📈 How It Works

The model learns the relationship between:

Hours Studied → Marks Obtained

Enter the number of study hours and receive an instant prediction.
